Este projeto consiste em uma **calculadora de IMC (Índice de Massa Corporal)** desenvolvida com o objetivo de treinar o **redirecionamento de telas em aplicações mobile**. 
O usuário informa seu peso e altura, o sistema realiza o cálculo do IMC utilizando a fórmula *IMC = peso / (altura²)* e, com base no resultado obtido, classifica o índice 
(como abaixo do peso, peso normal, sobrepeso ou obesidade). Após a classificação, o aplicativo **redireciona automaticamente o usuário para uma tela específica**, 
exibindo a mensagem correspondente à sua categoria, permitindo assim praticar lógica de decisão e navegação entre diferentes telas dentro da aplicação.
